Joke Generator!



1. The API I found generates a joke upon being requested. It contains the [setup], [punchline], and [type]. That last parameter is the extension I decided to impliment to the quote API of this assignment--assigning a category to whatever is pulled, in this case, the joke.



NOTE: A LOT of the jokes this API uses fall under "general". It's just the way it seems to have been made, so it might take a few clicks to get one from the other categories.



2. The API in my project would probably take the form of a mockup post generator, for the sake of simulating what an "explore" page would look like.
